Data Processing Addendum - pilot

Sablon operational pentru clienti B2B. Necesita review juridic si completare cu datele operatorului, imputernicitului, subprocesatorilor si masurilor tehnice.

Roluri

Clientul este operator pentru datele introduse in workspace si pentru workflow-ul intern. Thor actioneaza ca imputernicit pentru datele clientului si ca operator separat pentru securitate, billing si audit operational.

Masuri tehnice

Acces pe baza de rol, separare tenant, RLS in Supabase, criptare in tranzit, secrete in Cloudflare, audit events si control operational pentru surse.

Subprocesatori

Subprocesatorii pilot includ Cloudflare, Supabase, GitHub si Stripe daca billingul este activat.
